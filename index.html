PFO 2 - Sistema de Gestión de Tareas con API y Base de Datos

Proyecto desarrollado para la asignatura Programación sobre Redes. Este proyecto implementa una API REST con Flask, autenticación básica, contraseñas hasheadas y persistencia de datos en SQLite.

Endpoints principales

  • POST /registro: registra usuarios con contraseña hasheada.
  • POST /login: valida credenciales e inicia sesión.
  • GET /tareas: muestra una página HTML protegida.

Tecnologías utilizadas

  • Python
  • Flask
  • SQLite
  • Werkzeug Security
  • Requests
